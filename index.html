Synthetica 3.0

Synthetica is a Java program that is used to synthesize, analyze and simulate spatial linkages. Its' open architecture allows advanced users to add Java code to the system. Currently it runs on Windows, MacOSX, Linux and is based on OpenGL. Please visit the document page to download user's guide.
